Cómo jugar

Objetivo: reduce tu contaminación a 0 antes que el rival. Si se acaba el tiempo, gana quien tenga menos contaminación.

  • Empiezas con 50 puntos de contaminación.
  • Robas una carta al inicio de cada turno (excepto el primero).
  • Arrastra una carta a tu tablero para jugarla.
  • Cada carta tiene un efecto distinto que puede alterar la contaminación propia o rival.
